vnDataEndpoint

Description

This is a constant with two base urls for generating firebase or api access. Note in the code the file is named vn-data-endpoints (because there are more than one) But in code I use it as thus: vnDataEndpoint.fbUrl

Example

  • vnDataEndpoint.fbUrl -> https://brilliant-fire-5600.firebaseio.com
  • vnDataEndpoint.apiUrl -> http://www.samplestore.io/api/v1
