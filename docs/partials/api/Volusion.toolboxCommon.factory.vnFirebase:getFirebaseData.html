getFirebaseData
function in module Volusion.toolboxCommon

Description

The getFirebaseDataFn is exposed as a public api via getFirebaseData. Given a string as an endpoint it generated a path to the correct Firebase resource or it returns false. It depends on the vnConfig.getAccount() function to be correctly configured with the current account (as it appears in the Firebase schema) and have an appropriate secure token generated with the firebase-token-generator service (not an angular service). NOTE: the firebase- token-generator is still TODO.

Usage

getFirebaseData();
